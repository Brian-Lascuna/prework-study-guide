HTML

  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user
  • There are many different tags to help structure your webpage such as 'div' and 'span'

CSS

  • inline CSS: CSS rules applied within an HTML tag
  • internal CSS style sheet: CSS rules applied the 'head' tag
  • external CSS style sheet: CSS rules written on an external file and linked to the HTML
  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the inside of an element
  • '*' selector applies CSS rules to all elements on a page
  • '.' selector indicates a class

Git

  • git add -A: stages files to push into repository
  • git checkout -b branch-name: creates a new branch and switches to it
  • git status: checks what branch we are currently on
